MD File for CLI commands. 

- Tabs as auto complete. 
- Directory is another name for a folder. 

# `cd` Change Directory
- Used to change directories. 
## `cd . .` Directory rigth above this one. 
- Goes to the parent directory of the current directory
## `cd -` Directory used rigth before this one
- Puts us into the last Directory we used. 
## `cd ~`
- `~` refers to home

# `ls`
- Lists things inside the directory you're currently in 
## `ls -F` 
- Lists and differentiates the things inside directory.

# `pwd` Print working directory
- shows you the current directory you're in 

# `mkdir` Make directory
-  It will make a directory whereever the CLI is working currently. 

## `-p` 
- Allows us to make sub directories in a single `mkdir -p`command. 

# `touch` Creating files
- touch default.txt , will create a 0 byte empty text file named default

# `rm` removes/ deletes a file. 
- `-r` for removing directories
