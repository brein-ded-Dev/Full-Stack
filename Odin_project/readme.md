## Body
- <p></p> tag is used to write down paragreaphs. HTML will only provide a single space no matter how many spaces we put in the code itself. 
- <strong></strong> Used to bold text and to actually emphasise on them. Bold can be done by CSS but emphasis cannot be done on CSS
- <em></em> also emphasises on the text (italics), would recommend using it the same way as strong. 
- <a href="" target = "" rel="" ></a> The anchor tag is used to anchor one web page or image to another. 
Href is the hypertext reference, the link where the hyper text will take us to goes here. 

    Target - the tab where the href link opens like either the same tab or a new tab or etc. 
    -  `_self` = opens link in current tab
    - `_blank `= opens link in new tab
    - REL - it is used to define the relation betweeen the current tab and the tab opened by the href
        `noreferrer` - prevents certain detaills about the og page to pass on to the new page while preventing new page from accessing the og page. 
        `noopener`- prevents new website from accessing the og website
